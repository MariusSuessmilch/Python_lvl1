# Windsurf Editor - Anleitung für Programmieranfänger

Diese Anleitung zeigt dir die wichtigsten Funktionen von Windsurf, die du zum Programmieren lernen brauchst.

## 1. **Ersten Überblick bekommen**

Wenn du Windsurf öffnest, siehst du drei Hauptbereiche:
- **Links**: Der Datei-Explorer (deine Ordner und Dateien)
- **Mitte**: Der Code-Editor (hier schreibst du deine Programme)
- **Rechts**: Der Cascade Chat (dein AI-Programmierhelfer)

## 2. **Projekt/Ordner öffnen**

- Klicke auf **"Open Folder"** in der Mitte
- Wähle einen Ordner aus, in dem du programmieren möchtest
- Oder erstelle einen neuen Ordner für dein Projekt

## 3. **Dateien finden und erstellen**

**Datei-Explorer (links):**
- Hier siehst du alle deine Dateien und Ordner
- **Neue Datei erstellen**: Rechtsklick im Explorer → "New File"
- **Neue Python-Datei**: Erstelle eine Datei mit der Endung `.py` (z.B. `mein_programm.py`)

## 4. **Cascade Chat nutzen (dein AI-Helfer)**

**Chat öffnen/schließen:**
- **Öffnen**: Klicke auf das Chat-Symbol rechts oder drücke `Cmd/Ctrl + Shift + C`
- **Schließen**: Klicke auf das X im Chat-Bereich

**So nutzt du den Chat:**
- Stelle Fragen wie: "Wie schreibe ich eine Schleife in Python?"
- Bitte um Hilfe: "Kannst du mir bei diesem Fehler helfen?"
- Der AI kann sogar Code direkt in deine Dateien schreiben!

## 5. **Terminal verwenden**

**Terminal öffnen/schließen:**
- **Öffnen**: Klicke unten auf "Terminal" oder drücke `Cmd/Ctrl + J`
- **Schließen**: Klicke auf das X im Terminal-Bereich

**Python-Programme ausführen:**
Zu Python_Level_1 navigieren:
```bash
cd ~/Code/Python_Level_1```
```
Arbeitsumgebung aktivieren:
```bash
source .venv/bin/activate
```

```bash
python3 mein_programm.py
```

- Tippe den Befehl ein und drücke Enter
- **Tipp**: Nutze die **Tab-Taste** für automatische Vervollständigung von Dateinamen!

## 6. **Grundlegende Arbeitsschritte**

1. **Datei erstellen** → Python-Datei (`.py`) im Datei-Explorer erstellen
2. **Code schreiben** → Im mittleren Editor dein Programm schreiben
3. **Speichern** → `Cmd/Ctrl + S` (sehr wichtig!)
4. **Ausführen** → Im Terminal: `python dateiname.py`
5. **Bei Problemen** → Cascade Chat um Hilfe fragen

## 7. **Nützliche Tastenkombinationen**

- **Speichern**: `Cmd/Ctrl + S`
- **Rückgängig**: `Cmd/Ctrl + Z`
- **Kopieren**: `Cmd/Ctrl + C`
- **Einfügen**: `Cmd/Ctrl + V`
- **Terminal öffnen**: `Cmd/Ctrl + J`
- **Chat öffnen**: `Cmd/Ctrl + Shift + C`

## 8. **Tipps für den Anfang**

- **Speichere oft!** Gewöhne dir an, regelmäßig `Cmd/Ctrl + S` zu drücken
- **Nutze den Chat** - er ist dein bester Freund beim Lernen
- **Tab-Taste im Terminal** - spart viel Tipparbeit
- **Aussagekräftige Dateinamen** - z.B. `rechner.py` statt `test.py`

## 9. **Typischer Arbeitsablauf**

1. Ordner öffnen
2. Neue `.py`-Datei erstellen
3. Code schreiben (mit Hilfe des Chats)
4. Speichern
5. Terminal öffnen
6. Programm ausführen: `python dateiname.py`
7. Bei Fehlern: Chat fragen oder Code korrigieren

**Viel Erfolg beim Programmieren lernen!** 🚀
